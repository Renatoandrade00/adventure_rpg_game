Este projeto foi desenvolvido com o objetivo principal de **aprendizado pessoal** e exploração de tecnologias modernas para desenvolvimento de jogos web e integração Full-Stack (Frontend + Backend + Banco de Dados).

## 🚀 Sobre o Projeto

O jogo foi construído utilizando a engine **Phaser 3** integrada com o **Vite** para o Frontend, e um servidor **Node.js** com **MongoDB** para o Backend (salvamento de dados na nuvem). 

Durante o desenvolvimento, passamos por diversas fases e alcançamos as seguintes metas:

### ✅ Metas Alcançadas

- **Infraestrutura Full-Stack:** Conexão de um cliente Web (Phaser) com um servidor Express, persistindo contas de usuário de forma segura.
- **Sistema de Autenticação:** Tela de Login e Registro nativa no DOM do HTML antes de instanciar o Canvas do jogo.
- **Mundo em Blocos (Tilemap):** Criação de um mapa expansivo de 1600x960, dividindo uma área segura (Vila) e uma área de combate (Floresta).
- **Zonas Interativas:** Edifícios no mapa (Loja e Hospital) que transportam o jogador para "Cenas" diferentes.
- **Sistema de Combate e Turnos:** Batalhas de RPG clássicas com menu de ações (Atacar, Poção, Fugir) e inteligência artificial básica para os inimigos.
- **Economia e Progresso:** Sistema de Ouro (Drop dos monstros) e acúmulo de Experiência (XP).
- **Inteligência Artificial em Assets:** Todos os gráficos do jogo (Guerreiro, Arqueiro, Mago, Slime, Enfermeira, Mercador e Edifícios) foram gerados utilizando IA generativa, processados para fundo transparente (Alpha) via scripts customizados.
- **Deploy em Nuvem:** Hospedagem do banco de dados no MongoDB Atlas, e deploy do Frontend e Backend através do Render.com.

## 🛠️ Tecnologias Utilizadas

**Frontend:**
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) (Bundler rápido)
- [Phaser 3](https://phaser.io/) (Engine do Jogo)

**Backend:**
- [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) & Mongoose (Banco de dados na nuvem)
- `crypto` para Hashing de senhas
